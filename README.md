# Cuisinefy

**Cuisinefy** is a recipe web app that allows users to search for recipes based on ingredients, dish names, or cuisines. The app retrieves real-time recipe data from an API and displays detailed instructions on how to make the dish.

## Features

- Search for recipes by entering a dish name or ingredients.
- View detailed recipe information including ingredients and cooking instructions.
- Responsive and user-friendly design for a seamless experience.

## Technologies Used

- **HTML**: Structure of the web app.
- **CSS**: Styling and design.
- **JavaScript**: App functionality, including fetching data from the API.
- **API**: Integration with a recipe API for real-time data retrieval.
